# AI SDK for Manufacturing - Learning Guide

Welcome to the comprehensive guide for building AI-powered manufacturing applications using the Vercel AI SDK. This course focuses on practical, real-world applications in steel manufacturing.

## Course Overview

This course teaches you how to build three types of AI applications for manufacturing:

1. **Invisible AI** - Automate data extraction and forecasting without user interaction
2. **RAG (Retrieval-Augmented Generation)** - Build knowledge bases with semantic search
3. **Conversational AI** - Create interactive chatbots with tool calling

Each lesson builds on the previous one, teaching you progressively more advanced AI SDK features.

## Prerequisites

Before starting, make sure you have:

- [ ] **JavaScript/TypeScript knowledge** - Comfortable with modern JavaScript and TypeScript
- [ ] **React familiarity** - Understand React hooks and components
- [ ] **Node.js v20 or later** - LTS version recommended
- [ ] **pnpm package manager** - Install with `npm install -g pnpm`
- [ ] **Vercel account** - Free account at [vercel.com](https://vercel.com)
- [ ] **Git** - For cloning and version control
